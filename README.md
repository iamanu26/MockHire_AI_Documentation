## What This Repo Is

This repository contains the **static documentation site** for the MockHire AI project — a visual, single-page technical writeup built with vanilla HTML, CSS, and JavaScript.



---

## What's Inside

```
mockhire-ai-docs/
├── index.html      # Main documentation page
├── style.css       # All styling — custom design system
└── script.js       # Scroll animations, terminal demo, interactions
```

The page covers six sections:

| Section | Content |
|---------|---------|
| **Objective** | Problem statement, mission, and platform goals |
| **Features** | The four core modules explained |
| **Architecture** | Layered system diagram (Frontend → API → AI → DB) |
| **Interview Flow** | Step-by-step voice interview walkthrough |
| **Tech Stack** | Full breakdown of technologies used |
| **Engineering** | Key bugs found and solved during development |

## Running Locally

No build step needed. Just open the file:

```bash
# Clone this repo
git clone https://github.com/iamanu26/mockhire-ai-docs.git
cd mockhire-ai-docs

# Open directly in browser
open index.html
```

Or use any static server:

```bash
npx serve .
# or
python -m http.server 8080
```

## Design Notes

The docs site uses a custom "mission control" aesthetic — dark background, monospace type, neon accent colors, and SVG architecture diagrams built by hand. No frameworks, no dependencies. All layout and animation is plain CSS and vanilla JS.

Fonts used: [Space Mono](https://fonts.google.com/specimen/Space+Mono), [Syne](https://fonts.google.com/specimen/Syne), [DM Sans](https://fonts.google.com/specimen/DM+Sans) — all via Google Fonts.
